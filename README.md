# FRONTEND - Đồ án tốt nghiệp

1. **Create base project by react + vite**

```bash
npm create vite@latest my-react-app -- --template react
```

2. **Install dependencies**

```bash
npm install
```

3. **Run project**

```bash
npm run dev
```

4. **Build project**

```bash
npm run build
```
```bash
npx serve -s dist
```
